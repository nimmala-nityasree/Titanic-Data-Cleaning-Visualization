# Titanic Data Cleaning and Visualization

## Project Overview

This project demonstrates data cleaning and visualization using the Titanic dataset.

## Tools Used

* Python
* Pandas
* Matplotlib

## Tasks Performed

* Loaded and explored the dataset
* Identified missing values
* Filled missing values in the Age column
* Created visualizations for data analysis

## Visualizations

1. Survival Count Bar Chart
2. Gender Distribution Pie Chart
3. Age Distribution Histogram

## Key Insights

* More passengers did not survive than survived.
* The dataset contains more male passengers than female passengers.
* Most passengers were between 20 and 40 years old.
